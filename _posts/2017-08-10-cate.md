<h2>cate</h2>
convert android studio project to eclipse workspace

可以将Android Studio中的工程转为Eclipse可以导入的项目结构

```
cate	英[keɪt]
美[keɪt]
n.	<古>佳肴，美食;
[其他]	复数：cates

```

<h2>Usage</h2>
```
usage: cate [Android Studio Project Path] [Destination Path]
```
