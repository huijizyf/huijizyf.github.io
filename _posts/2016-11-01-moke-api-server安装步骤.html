moke-api-server安装步骤
1.安装Python2.7
2.安装Node.js4.4
3.安装npm
4.安装libcurl4-openssl-dev
5.安装mock-api-server
通过mock-api-server --port  PORT     可以启动mock-api-server
注意：
    启动mock-api-server时，切换到自建的指定目录，然后按需要建立目录结构
例：
在目录~/mock-api下启动server
目录结构为~/mock-api/test/mock-api/GET/hello.json
访问http://ip/port/hello时，返回的数据为hello.json文件中的内容
